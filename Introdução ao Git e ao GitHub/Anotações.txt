Git bash é um terminal extendido para otimizar o uso do Git.

código para clonar repositório do GitHub: git clone (e link do repositório);

git status - (Para ver o status da pasta atual);

git add . ou git add -A (para adicionar os arquivos novos que não estavam incluídos no commit anterior);

para commitar usa-se o código (git commit -m) "o -m é usado para colocar uma mensagem logo em seguida entre aspas";

para mandar de volta para o GitHub, basta usar o código: (git push origin main);
